# Prompts

Prompts are the core unit of interaction in Dataelan, allowing users to execute AI models across different modalities.

## Prompt Sessions

A prompt session is a collection of related prompts that share context and settings.

### Session Types

1. **TEXT**
   - Natural language processing
   - Text completion
   - Question answering
   - Summarization

2. **CODE**
   - Code generation
   - Code review
   - Bug fixing
   - Documentation

3. **IMAGE**
   - Image generation
   - Image editing
   - Style transfer
   - Vision analysis

4. **VOICE**
   - Speech-to-text
   - Text-to-speech
   - Voice cloning
   - Audio analysis

5. **VIDEO**
   - Video generation
   - Video editing
   - Motion analysis
   - Scene understanding
